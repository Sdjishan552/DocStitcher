How to use

  1. Add a two-sided or one-sided document card using the buttons below.
  2. Upload front (and back for two-sided). PDFs are auto-converted to images.
  3. Enable the signature checkbox to add a signature image. Use the 🎯 Place Signature button to position it precisely using arrow keys (hold Shift for bigger steps).
  4. Click Download and choose PDF, JPG, or PNG output format.
  5. Use Single-Doc Tools (bottom right) to crop, rotate, flip, compress and convert any image or PDF all at once before sending it here.
