Document Stitcher

A4 only - local PDF maker - no uploads

How to use

  1. Add one document card for every front/back pair.
  2. Upload the front and back images. The signature image is optional.
  3. Choose portrait for stacked placement or landscape for left/right placement.
  4. Adjust each card's quality slider and watch its PDF size update.
  5. Download cards separately or use Download All to start each PDF download one by one.
